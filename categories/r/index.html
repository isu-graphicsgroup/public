blogdown quick tutorial

Wed Feb 21, 2018 by Weicheng Zhu in  R

In this talk, I will show you how to create a professional academic homepage (or blog) from scratch using the R package ‘blogdown’. I will first introduce some basic concepts of web construction and some essential tools (‘rmarkdown’ and ‘git/github’) that will be used in building websites with ‘blogdown’. Then I will illustrate how to build a website step by step. By the end of the meeting, you are expected to have your own homepage created! Read more →

Calendar-based graphics for visualizing people’s daily schedules

Thu Feb 08, 2018 by Earo Wang in  R

Calendars are broadly used in society to display temporal information, and events. This paper describes a new R package with functionality to organize and display temporal data, collected on sub-daily resolution, into a calendar layout. The function frame_calendar uses linear algebra on the date variable to restructure data into a format lending itself to calendar layouts. The user can apply the grammar of graphics to create plots inside each calendar cell, and thus the displays synchronize neatly with ggplot2 graphics. Read more →
